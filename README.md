# Dusk till Dawn Station

Retro-synthwave live TV station playing public domain animation classics. 

## Broadcast Hours
- **On Air:** 6:00 PM – 6:00 AM (Local Time)
- **Off Air:** 6:00 AM – 6:00 PM (Local Time)

## Features
- Retro 4:3 CRT TV frame with scanline overlays.
- Live-synced broadcast: All visitors see the same frame of the same video.
- Public domain classics: Steamboat Willie, The Skeleton Dance, and more.
